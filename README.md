## 💧 Su Takip - Kişisel Su Asistanı

Dart ve Flutter ile geliştirilmiş, kullanıcıların günlük su içme alışkanlıklarını düzenlemelerine yardımcı olan, modern tasarımlı ve kullanıcı dostu bir mobil uygulama.

## Özellikler

* **Dinamik Görselleştirme:** İçilen su miktarına göre gerçek zamanlı dolan animasyonlu termos tasarımı.
* **Kişiselleştirilmiş Hedefler:** Kullanıcının ihtiyacına göre güncelleyebileceği dinamik günlük su hedefi belirleme.
* **Akıllı Motivasyon Sistemi:** İlerleme durumuna göre kullanıcıyı teşvik eden dinamik mesajlar ve asistan desteği.
* **Kutlama Efektleri:** Günlük hedefe ulaşıldığında `Confetti` efekti ile kullanıcıyı ödüllendirme.
* **Kalıcı Veri Depolama:** `Shared Preferences` entegrasyonu sayesinde uygulama kapatılsa bile verilerin korunması.
* **Gelişmiş Kontroller:** Yanlış girişler için "Geri Al" özelliği ve yeni güne başlarken "Sıfırla" seçeneği.
* **Modern UI/UX:** Google Fonts (Poppins) ve özel tasarım widget'lar ile ferah bir kullanıcı deneyimi.

## Kullanılan Teknolojiler

* **Framework:** [Flutter](https://flutter.dev)
* **Programlama Dili:** [Dart](https://dart.dev)
* **State Management:** [Provider](https://pub.dev/packages/provider) (Uygulama genelinde veri yönetimi için)
* **Yerel Veritabanı:** [Shared Preferences](https://pub.dev/packages/shared_preferences) (Kalıcı veri saklama için)
* **Animasyon:** [Confetti](https://pub.dev/packages/confetti) (Hedef kutlamaları için)
* **Tasarım:** Custom Widgets & Google Fonts

## Kurulum

1.  Bu depoyu klonlayın:
    ```bash
    git clone [https://github.com/kullaniciadin/su-takip.git](https://github.com/kullaniciadin/su-takip.git)
    ```
2.  Proje dizinine gidin:
    ```bash
    cd su-takip
    ```
3.  Paketleri yükleyin:
    ```bash
    flutter pub get
    ```
4.  Uygulamayı çalıştırın:
    ```bash
    flutter run
    ```
